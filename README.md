Тестовое задание для Ketopay
Реализация на React, Vite
Достать список новостей по API  и отрисовать под мобильную вёрстку по дизайну. Загружать можно до любой даты помесячно


## Запуск 
Заходим в папку nuxt или next

В дев режиме:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

В продакшен режиме:

```bash
npm run build
npm run preview
# or
yarn run build
yarn run preview
# or  
pnpm run build  
pnpm run preview
# or
bun run build
bun run preview
